Images

Images can be included in HTML documents using the <img> tag. The src attribute specifies the path to the image file, and the alt attribute provides alternative text for screen readers and when the image cannot be displayed.

Example

<img src="path/to/image.jpg" alt="Description of the image">

Attributes

  • src: Specifies the URL of the image.
  • alt: Provides alternative text for the image.
  • width: Sets the width of the image (in pixels or percentage).
  • height: Sets the height of the image (in pixels or percentage).

Example with Width and Height

<img src="path/to/image.jpg" alt="Description of the image" width="300" height="200">
