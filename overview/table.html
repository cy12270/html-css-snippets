Table

HTML tables are used to display data in a tabular format using rows and columns. The main elements used to create a table are:

  • <table>: Defines the table.
  • <tr>: Defines a table row.
  • <th>: Defines a table header cell.
  • <td>: Defines a table data cell.

Here is a basic example of an HTML table:

<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>

This code will create a table with two columns and two rows, where the first row contains header cells and the second row contains data cells.

Header 1 Header 2
Data 1 Data 2

With caption:

Sample Table Caption
